投稿JSON格式

游戏投稿JSON格式

请按照以下格式提交游戏投稿，发送到投稿邮箱：artemisgames@xbmail.dpdns.org

数据结构

游戏数据应为一个JSON数组，每个游戏是一个对象

[ { "id": 1, "name": "游戏名称", "image": "https://example.com/game-image.jpg", "port_author": "移植作者", "install_author": "直装作者", "link": "https://example.com/download/1" }, { "id": 2, "name": "另一个游戏", "image": "https://example.com/another-game.jpg", "port_author": "移植作者", "install_author": "直装作者", "link": "https://example.com/download/2" } ]

字段说明

id: 无需填写

name: 游戏名称，字符串类型

image: 游戏封面图URL，若有则填写

port_author: 移植作者，字符串类型

install_author: 直装作者，字符串类型

link: 游戏下载链接，字符串类型

友链投稿JSON格式

请按照以下格式提交友情链接投稿，发送到投稿邮箱：artemislinks@xbmail.dpdns.org

数据结构

友情链接数据应为一个JSON数组，每个链接是一个对象

[ { "name": "网站名称", "owner": "站长名称", "url": "https://example.com" }, { "name": "另一个网站", "owner": "另一位站长", "url": "https://anotherexample.com" } ]

字段说明

name: 网站名称，字符串类型

owner: 站长名称，字符串类型

url: 网站链接，字符串类型
